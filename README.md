# Swf2MuteMp4-python

1. 安装 Python pylzma 模块
  pip install pylzma
  
2. 安装gnash flash 播放器
  sudo apt-get install gnash
  
3. 安装 ffmpeg
  sudo add-apt-repository ppa:kirillshkrogalev/ffmpeg-next
  sudo apt-get update
  sudo apt-get install ffmpeg
  
  
#使用方法
  打开终端，cd到对应的目录下，运行main.py文件
  main.py input.swf
